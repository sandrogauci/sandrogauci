# Hi, I'm Sandro Gauci

I run [Enable Security](https://www.enablesecurity.com), where we focus on security for real-time communications (VoIP and WebRTC). We do penetration testing, fuzzing, code review, and DDoS resilience testing for RTC systems.

## Projects

- [SIPVicious](https://github.com/enablesecurity/sipvicious) - VoIP/SIP security testing toolset (1k+ stars)
- [Awesome RTC Hacking](https://github.com/EnableSecurity/awesome-rtc-hacking) - curated list of VoIP, WebRTC, and VoLTE security resources
- [WAFW00F](https://github.com/enablesecurity/wafw00f) - Web Application Firewall fingerprinting tool (6k+ stars)

## Writing and research
